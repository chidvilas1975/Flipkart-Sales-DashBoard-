üìà Key Performance Indicators (KPIs)

| KPI                               | Description                                                          |
| --------------------------------- | -------------------------------------------------------------------- |
| **Total Sales**                   | Represents the overall revenue generated during the analyzed period. |
| **Total Quantity Sold**           | Indicates the total number of units sold across all products.        |
| **Average Order Value (AOV)**     | Measures the average value of each customer order.                   |
| **Profit Margin**                 | Displays the profitability percentage for sales transactions.        |
| **Top Performing Categories**     | Highlights categories contributing most to total sales.              |
| **Top Performing States/Regions** | Shows which locations are generating the highest sales.              |
| **Customer Count**                | Tracks the number of unique customers who made purchases.            |
| **Monthly/Quarterly Trend**       | Visualizes how sales and profit vary over time.                      |
